# LUMISTYLE Monorepo

This repository contains both the frontend React application and the backend API server for LUMISTYLE.

## Setup and Running Locally

To run both the frontend and backend concurrently:

1. Install dependencies at the root and for both projects:
   ```bash
   npm run install:all
   ```
2. Start both servers in development mode:
   ```bash
   npm run dev
   ```

## Folder Structure

* `backend/`: Node.js Express server running on port 5000.
* `frontend/`: React Vite frontend running on port 5173.
